# Insert-Coin script for Mister FPGA

This script creates an "_Insert Coin" directory in _Arcade folder, with all arcade games sorted by System and Editor.
It also adds several cores (SEGASYS1 and NEOGEO from black wine)
It will install lastest console builds (PSX, Saturn, SGB, S32X)
It will also replace Main Mister with a version removing the rom loading progress bar (this version is updated once a week at least)

To use it download complete_scripts.7z and unzip in your Script folder. ie : /media/fat/Scripts/insertcoin
then run insertcoin.sh script.
To update the menu, just run this script again.

The script relies on update_all script 
https://github.com/theypsilon/Update_All_MiSTer
